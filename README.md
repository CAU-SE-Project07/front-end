# 프론트엔드
중앙대학교 소프트웨어공학 2024 봄학기 텀프로젝트
### ✅ 프로젝트 소개
---
**이슈 관리 시스템 개발 프로젝트**  
소프트웨어 개발 과정에서 조직/단체의 필요에 의해 이슈 목록과 내용 등을 체계적으로 관리하는 소프트웨어의 일종으로 트러블 티켓 시스템(trouble ticket system)이라고 함.
